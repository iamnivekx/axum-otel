# Contributing to axum-otel

We welcome contributions to axum-otel! We appreciate your help in making this project better.

## How to Contribute

- **Reporting Bugs or Requesting Features:** If you find a bug or have a feature request, please open an issue on the GitHub issue tracker. Provide as much detail as possible, including steps to reproduce for bugs.
- **Pull Requests:** We actively welcome your pull requests.
    - Fork the repository and create your branch from `main` (or the current default branch).
    - Please ensure your code adheres to the existing style.
    - Ensure that tests pass (if applicable). Add new tests for new functionality.
    - Make sure your commit messages are clear and descriptive.
    - Please feel free to submit a Pull Request.

Thank you for your contributions!
